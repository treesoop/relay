만든 게 있어서 테스트해볼 사람 찾는 중. 2주만.

**Relay** — Claude Code에서 방금 삽질한 문제를 skill로 저장해서 나중에 나도 쓰고 다른 사람도 쓰게 하는 commons 레이어.

    curl -fsSL https://raw.githubusercontent.com/treesoop/relay/main/install.sh | bash
